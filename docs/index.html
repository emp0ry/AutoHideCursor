Windows tray app · Auto-hide cursor · No admin required

Hide the cursor. Keep the focus.

AutoHide Cursor is a lightweight Windows tray app that automatically hides the mouse cursor after inactivity. Set a custom timeout, toggle it from the tray, change the tray icon, and control auto-start, all running as a standard user with no admin needed.

Features

Out of sight, until you move.

A standard Windows low-level mouse hook and system cursor swap keep the cursor hidden while you read, watch, or present, then bring it right back the moment you move.

Auto-hide cursor

The cursor disappears automatically after a period of inactivity (10s by default).

Configurable timeout

Cycle between 3, 5, 10, 30, and 60 seconds straight from the tray menu.

Quick toggle

Left-click the tray icon to enable or disable auto-hide globally.

Custom tray icon

Load any .ico; the default icon auto-adapts to taskbar light/dark mode.

Persistent & auto-update

Timeout, enabled state, and custom icon are restored on launch; updates are checked on start.

Run at startup

Enable or disable auto-start from the tray menu (HKCU only), no admin required.

Usage

Everything from the tray.

Launch AutoHideCursor.exe and it appears as an icon in the notification area. Left-click toggles it; right-click opens the menu.

01

Enable or disable

Left-click the tray icon to toggle AutoHide Cursor on or off.

02

Change the timeout

Right-click → Timeout: Xs to cycle through 3s → 5s → 10s → 30s → 60s.

03

Personalize

Right-click → Run at Startup, Change Icon / Reset Icon, or About.

Requirements

Standard APIs, no elevation.

AutoHide Cursor uses only standard Windows APIs (a low-level mouse hook and a system cursor swap) and runs entirely as a standard user.

Windows 10 & 11

Windows 10 22H2 or Windows 11 23H2/24H2 (and newer).

No admin

No administrator privileges required to install or run.

Install & run

Download from Releases, extract if zipped, and launch AutoHideCursor.exe.

Uninstall

Exit from the tray and delete the files. Turn off Run at Startup first if enabled.
